Install Guide

安装插件

支持 Chrome MV3。若已上架商店，可直接在商店页面安装。

本地加载安装（开发者模式）

  1. 打开 Chrome → 地址栏输入 chrome://extensions/
  2. 右上角开启「开发者模式」
  3. 点击「加载已解压的扩展程序」
  4. 选择项目根目录：Vocabulary Builder
  5. 安装完成后，浏览网页划词即可使用

商店安装（如已上架）

替换为 Chrome Web Store 的安装链接。

打开商店页面
如果你是企业环境，可能需要允许扩展安装权限。
安装后如需升级专业版，请前往购买页面获取 License Key，再回到激活页面完成绑定。
